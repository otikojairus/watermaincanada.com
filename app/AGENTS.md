# Water Main Canada Route Notes

This app uses a new `wmc-*` visual system and spreadsheet-driven pSEO pages.

- Landing page source data lives in `lib/watermain-data.ts`.
- Dynamic city/service pages are served from `app/[slug]/page.tsx`.
- Use one primary keyword target per landing page.
- Keep copy visitor-facing: describe symptoms, safety, inspection, and next steps.
- Preserve JSON-LD for LocalBusiness, Service, FAQPage, and BreadcrumbList.
